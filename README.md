# COMPUTEX 2026 Interactive Guide

這是一套可直接部署到 GitHub Pages 的靜態網站。

## 檔案
- `index.html`：網站入口
- `styles.css`：深色視覺與響應式版型
- `data.js`：展會資料
- `app.js`：互動功能

## 使用方式
1. 在 GitHub 建立 repo，例如 `computex-2026-digital-guide`
2. 上傳本資料夾內所有檔案到 repo 根目錄
3. 到 Settings → Pages
4. Source 選 `Deploy from a branch`
5. Branch 選 `main` / root
6. 儲存後等待 GitHub Pages 產生網址

## 互動功能
- 首頁 Overview
- 5 大路線切換
- 廠商搜尋與篩選
- 廠商詳情彈窗
- 收藏清單 Saved
- Keynote Timeline
- Analyst Mode
- 可列印觀展筆記
